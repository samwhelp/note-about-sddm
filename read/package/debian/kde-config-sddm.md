## 安裝指令

執行下面指令，安裝「Package: [kde-config-sddm](https://packages.debian.org/stable/kde-config-sddm)」。

``` sh
sudo apt-get install kde-config-sddm
```

## bin

執行下面指令，找出相關的指令

``` sh
dpkg -L kde-config-sddm | grep bin
```

顯示

```
/usr/bin
/usr/bin/sddmthemeinstaller
```


## applications

執行下面指令，找出相關的「Desktop Entry」

``` sh
dpkg -L kde-config-sddm | grep 'applications' | sort -u
```

顯示

```
/usr/share/applications
/usr/share/applications/kcm_sddm.desktop
```

執行下面指令，找出「/usr/share/applications/kcm_sddm.desktop」所執行的指令。

``` sh
grep '^Exec' /usr/share/applications/kcm_sddm.desktop
```

顯示

``` ini
Exec=systemsettings kcm_sddm
```

也就是可以在「Terminal」執行「`systemsettings kcm_sddm`」。
